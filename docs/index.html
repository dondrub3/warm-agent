简单易用的API

RESTful API设计，快速集成到您的应用中

POST /v1/emotion/analyze 
curl -X POST https://api.warm-agent.com/v1/emotion/analyze \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "今天工作压力好大",
    "user_id": "user_123"
  }'

# 响应示例
{
  "emotion": "压力",
  "confidence": 0.89,
  "suggested_response": "工作压力确实让人疲惫，适当休息一下会更好。",
  "needs_support": true
}
